About PetClinic

PetClinic is a web application developed using HTML, CSS, JavaScript, Node.js, Express and MySQL. The application allows pet owner registration, searching owner records and maintaining pet inventory. This project will be deployed on an Ubuntu EC2 instance and connected to an AWS RDS MySQL database.
